Stock Alert V6.9 Update System

เพิ่มระบบอัปเดตในแอป:
- หน้าแรกแสดง "เป็นเวอร์ชันล่าสุดแล้ว V6.9" เมื่อไม่มีอัปเดต
- ถ้าอัปโหลดเวอร์ชันใหม่ที่มี version.json เวอร์ชันสูงกว่า จะขึ้น "มีอัปเดตใหม่"
- เมื่อกดเข้าไปมีปุ่ม "อัปเดต" และ "ภายหลัง"
- กดอัปเดตแล้วแอปล้าง cache/service worker และโหลดไฟล์ล่าสุดจากเว็บเดิม

วิธีใช้ในอนาคต:
1. อัปโหลดไฟล์เวอร์ชันใหม่ขึ้นเว็บเดิม
2. แก้ version.json ให้ version สูงขึ้น เช่น 6.9
3. เครื่องพนักงานเปิดแอปแล้วจะตรวจพบอัปเดตเอง
